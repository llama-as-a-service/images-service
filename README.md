## API Documentation

### Random Endpoint

| URL            | /random |
|----------------|---------|
| Method         | GET     |
| Status         | 200 OK  |
| Authentication | None    |

#### Optional Parameters
Defualt = 1
Max = 25
```
?count=9
```

#### Sample Response:
```json
{
    "message": [
      "https://cdn.britannica.com/41/1/Alpaca.jpg?w=400&h=300&c=crop",
      "https://images.pexels.com/photos/3396661/pexe61.jpg&fm=jpg",
      "https://www.openaccess.org/wp-content/scaled.jpg",
      ...
    ],
}
```

### Upload Endpoint

| URL            | /upload     |
|----------------|-------------|
| Method         | POST        |
| Status         | 201 CREATED |
| Authentication | Admin       |

#### Example body:
```json
{
    "url": "https://cdn.britannica.com/41/1/Alpaca.jpg?w=400&h=300&c=crop",
}
```

#### Sample Response:
```json
{
    "message": "Successfully upload image!",
}
```
